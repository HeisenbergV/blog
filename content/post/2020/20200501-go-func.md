## 参数传递
** go的所有参数传递，都是值传递不存在引用传递。 ** 

## 接口实现
```go
type Cat struct {
}

func Hello(c Cat){
}

func main(){
    c := Cat{}
	Hello(c)
}
```

```go
type Cat struct {
}

func (c Cat) Hello() {
}

func main(){
    c := Cat{}
	c.Hello()
}
```
两种写法无任何区别：
1. 底层汇编都一样（只是函数名不一样）
2. 调用流程和结果都一样
只有在设计方面有区别
